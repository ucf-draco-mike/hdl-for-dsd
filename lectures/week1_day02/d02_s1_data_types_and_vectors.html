wire vs. reg — Misleading Names

wire

Combinational connection. Must be continuously driven by assign or module output.

reg

Can be assigned inside always blocks. Does NOT always mean register!

// This reg → flip-flop (clock edge)
reg [7:0] r_counter;
always @(posedge clk) r_counter <= r_counter + 1;

// This reg → combinational logic (no clock!)
reg [3:0] r_mux_out;
always @(*) r_mux_out = sel ? a : b;
wire must be driven by assign or module output. reg can be assigned inside always blocks but does NOT always become a register — it depends on whether there's a clock edge.

The Decision Flowchart

SystemVerilog fixes this confusion with the logic type — Week 4.
Follow this decision flowchart. Driven by assign or module port? Use wire. Assigned inside always? Use reg. Whether reg becomes a flip-flop depends on posedge clk. SystemVerilog simplifies this with logic.

Vectors (Buses)

wire [7:0] w_data;    // 8-bit bus: bit 7 = MSB, bit 0 = LSB
wire [3:0] w_nibble;  // 4-bit bus

w_data[7]      // single bit — the MSB
w_data[3:0]    // lower nibble — bits 3 down to 0
w_data[7:4]    // upper nibble — bits 7 down to 4
Convention: Always use [MSB:LSB] with LSB = 0. You can use ascending [0:7], but don’t.
Vectors are multi-bit signals. MSB:LSB notation. Bit indexing and slicing. Convention: always descending with LSB = 0.

Concatenation & Replication

// Concatenation: join signals
wire [7:0] w_byte = {w_nibble_hi, w_nibble_lo};  // 4+4 = 8 ✓

// Replication: repeat a pattern
wire [7:0] w_all_ones = {8{1'b1}};    // 8'b11111111

// Sign extension: replicate MSB to fill upper bits
wire [7:0] w_sign_ext = {{4{w_nibble[3]}}, w_nibble}; // 4→8
Sign extension is a pattern you'll use constantly. Double braces {{N{bit}}, signal} — it's just concatenation of a replication with a signal.
Concatenation joins signals with curly braces. Replication repeats a pattern. Sign extension replicates the MSB. The double curly braces are just concatenation of a replication.

Key Takeaways

① wire = driven by assign. reg = assigned in always.

② reg does NOT always mean register — it depends on context.

③ Vectors: [7:0] is 8 bits. Slice with [3:0]. Join with {a, b}.

④ Always use [MSB:LSB] with LSB = 0.
