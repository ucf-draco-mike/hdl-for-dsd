# Day 15: Final Project Build Day

## Session Structure (no pre-class videos)

| Time | Activity |
|---|---|
| 0:00–0:15 | Integration briefing (slides) |
| 0:15–0:45 | Individual check-ins |
| 0:45–2:00 | Build time with on-call support |
| 2:00–2:15 | Status round |
| 2:15–2:30 | Buffer / overflow debugging |

## Materials

| File | Description |
|------|-------------|
| `d15_briefing_slides.html` | 15-min integration strategy slides (shown live, ~8 slides) |
| `d15_integration_guide.md` | Student reference handout: checklist + common issues + demo prep |

## Key Themes
- No Big Bang Integration — add one module at a time
- Skeleton top module with heartbeat → synthesize first
- UART printf debugging for hardware-only issues
- Minimum viable demo: something must work on hardware by Day 16
- Individual check-ins: status, block diagram review, scope management
