Beyond the Free-Running Counter

Day 4 gave you a simple incrementing counter. Real designs need:

Modulo-N: counts 0 to N−1, then wraps (any period)

Up/Down: direction-controlled counting

Loadable: preset starting value (timers, address generators)

Day 4 gave you simple counters. Now we build the variations you'll actually use: modulo-N, up/down, loadable.

Modulo-N Counter

module counter_mod_n #(
    parameter N = 10
)(
    input  wire                  i_clk,
    input  wire                  i_reset,
    output reg  [$clog2(N)-1:0]  o_count,
    output wire                  o_wrap
);
    always @(posedge i_clk) begin
        if (i_reset)             o_count <= 0;
        else if (o_count == N-1) o_count <= 0;
        else                     o_count <= o_count + 1;
    end
    assign o_wrap = (o_count == N - 1);
endmodule
$clog2(N) = ceil(log₂(N)) — automatic width sizing. Change N and the bit width adjusts. No manual math.
Modulo-N wraps at any value. Dollar-clog2 auto-sizes the width. The wrap output pulses for one cycle at rollover — useful for cascading counters.

Key Takeaways

① Modulo-N: counts 0 to N−1, wraps. $clog2 auto-sizes width.

② Up/down: direction input controls increment/decrement.

③ Loadable: preset starting value via load enable.

④ Wrap/done output enables cascading and timing.
