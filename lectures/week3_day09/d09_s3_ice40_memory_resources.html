Key Takeaways

① iCE40 HX1K: 1,280 LUTs + 16 EBR × 4 Kbit = 64 Kbit block RAM.

② Sync read → EBR inference (saves LUTs). Async read → distributed (LUTs).

③ Each EBR configurable: 256×16, 512×8, 1024×4, 2048×2.

④ Budget your EBRs for the final project — 16 blocks is generous.
