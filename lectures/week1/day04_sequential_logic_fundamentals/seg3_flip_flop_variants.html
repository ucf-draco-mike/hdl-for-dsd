D Flip-Flop Variants

// Basic D flip-flop
always @(posedge i_clk)
    r_q <= r_d;

// With synchronous reset
always @(posedge i_clk)
    if (i_reset) r_q <= 1'b0;
    else         r_q <= r_d;

// With asynchronous reset
always @(posedge i_clk or posedge i_reset)
    if (i_reset) r_q <= 1'b0;
    else         r_q <= r_d;

// With enable
always @(posedge i_clk)
    if (i_reset)    r_q <= 1'b0;
    else if (i_en)  r_q <= r_d;
Four D flip-flop variants you'll use constantly. Basic — just captures on edge. Synchronous reset — resets on the clock edge. Asynchronous reset — resets immediately, note the sensitivity list includes reset. With enable — only captures when enable is high.

Our convention: synchronous, active-high reset unless there's a specific reason otherwise.
Synchronous reset is cleaner and we'll prefer it for this course. It resets on the clock edge, keeps timing predictable, and works well with the iCE40.

Key Takeaways

① Basic D-FF: always @(posedge clk) r_q <= r_d;

② Sync reset: if (reset) r_q <= 0; else r_q <= r_d;

③ Enable: adds another condition — else if (en)

④ Prefer synchronous, active-high reset on iCE40.
