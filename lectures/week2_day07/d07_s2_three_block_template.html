Block 1: State Register

// Block 1 — Sequential: just a flip-flop
always @(posedge i_clk)
    if (i_reset)
        r_state <= S_IDLE;
    else
        r_state <= r_next_state;
Block 1 is always this simple. No logic here — just load next state. If you’re putting if/case in Block 1, you’re doing it wrong.
Block 1 is always trivial — just a flip-flop that loads next state on clock edge. If you're putting logic here, you're doing it wrong.

Block 2: Next-State Logic

// Block 2 — Combinational: decide next state
always @(*) begin
    r_next_state = r_state;  // DEFAULT: stay put
    case (r_state)
        S_GREEN: begin
            if (w_timer_done)
                r_next_state = S_YELLOW;
        end
        S_YELLOW: begin
            if (w_timer_done)
                r_next_state = S_RED;
        end
        S_RED: begin
            if (w_timer_done)
                r_next_state = S_GREEN;
        end
        default: r_next_state = S_GREEN;
    endcase
end
The green line is critical: r_next_state = r_state prevents latches and handles “stay in current state” by default.
Block 2 computes next state. The critical first line: r_next_state = r_state. This default prevents latches and means 'if no transition fires, stay put'. Always include default case for illegal states.

Block 3: Output Logic

// Block 3 — Combinational: compute outputs from state (Moore)
always @(*) begin
    // Defaults — prevent latches
    o_red = 1'b0; o_yellow = 1'b0; o_green = 1'b0;
    case (r_state)
        S_GREEN:  o_green  = 1'b1;
        S_YELLOW: o_yellow = 1'b1;
        S_RED:    o_red    = 1'b1;
        default:  o_red    = 1'b1;  // safe default
    endcase
end
Moore: outputs depend only on r_state. Output defaults at the top prevent latches — same pattern as Day 3.
Block 3 maps state to outputs. Moore style: outputs depend only on r_state. Default assignments at top prevent latches. Same pattern as Day 3.

Key Takeaways

① Block 1 (seq): simple FF — r_state <= r_next_state.

② Block 2 (comb): next-state logic with r_next_state = r_state default.

③ Block 3 (comb): output logic with defaults — Moore: f(state) only.

④ This template works for any FSM. Memorize it.
