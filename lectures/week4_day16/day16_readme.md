| Time | Activity |

| 2:00–2:15 | Course retrospective (slides) |

## Materials

| File | Description |
|------|-------------|
| `d16_retrospective_slides.html` | Course wrap-up slides (shown live after demos, ~9 slides) |
| `d16_rubric.md` | Evaluation rubric for student demos (distribute on Day 15) |
| `d16_next_steps.md` | Career pathways, resources, and project ideas reference handout |

## Demo Format (5–7 min per student)

1. Introduction (30s): project name, one sentence
2. Live demo (2m): show it working on Go Board
3. Architecture (1–2m): block diagram walk-through
4. Verification (1m): testbench results, assertions
5. Reflection (1m): challenges, what you'd change
6. Q&A (remaining)

## Evaluation Weights

- Functionality: 30%
- Design Quality: 25%
- Verification: 20%
- Integration: 15%
- Presentation: 10%
