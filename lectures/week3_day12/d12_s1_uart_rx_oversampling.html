Why RX Is Harder Than TX

TX controls timing вҖ” it generates the baud clock. Easy.

RX must discover timing вҖ” data arrives asynchronously. When do you sample?

Solution: 16Г— oversampling. Sample at 16Г— the baud rate to find the center of each bit period.
TX has it easy вҖ” it controls timing. RX must figure out when to sample without a shared clock. Solution: sample at 16x the baud rate.

Start-Bit Detection

1. IDLE: RX line is high. Continuously oversample.

2. Falling edge detected: RX goes low вҖ” possible start bit.

3. Wait 7-8 oversample ticks вҶ’ center of start bit. Sample again.

4. Still low? Confirmed start bit! High? Glitch вҖ” return to IDLE.

5. From start center, count 16 ticks вҶ’ center of D0. Sample. Repeat for D1вҖ“D7.

6. Count 16 ticks вҶ’ center of stop bit. Verify high. Assert o_valid.

Start bit detection: falling edge triggers, wait to center, verify still low, then sample 8 data bits at 16-tick intervals. Verify stop bit.
