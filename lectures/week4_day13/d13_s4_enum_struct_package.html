enum — Named States

// Verilog
localparam S_IDLE  = 2'b00;
localparam S_START = 2'b01;
localparam S_DATA  = 2'b10;
localparam S_STOP  = 2'b11;
reg [1:0] r_state;
// SystemVerilog
typedef enum logic [1:0] {
    S_IDLE, S_START,
    S_DATA, S_STOP
} uart_state_t;
uart_state_t state;

Type safety: state = 3; → compile warning

Debug: $display("State: %s", state.name()); prints "S_IDLE"

Synthesis: Identical to localparam — zero hardware cost

Enum gives you named, type-safe states. Compiler warns on invalid assignments. state.name() prints readable names in simulation. Zero hardware cost.

struct packed — Grouped Signals

typedef struct packed {
    logic [7:0] data;
    logic       valid;
    logic       busy;
} uart_tx_ctrl_t;             // 10 bits wide

uart_tx_ctrl_t tx;

tx.data  = 8'h41;             // dot notation
tx.valid = 1'b1;
if (tx.busy) ...               // readable, clean
packed = contiguous bit vector. Fully synthesizable. Same as logic [9:0] but with named fields.
Packed struct: contiguous bit vector with named fields. Fully synthesizable, zero overhead. Groups related signals for cleaner interfaces.

package — Shared Definitions

// uart_pkg.sv — shared across all UART modules
package uart_pkg;
    typedef enum logic [1:0] {
        S_IDLE, S_START, S_DATA, S_STOP
    } uart_state_t;

    localparam int CLK_FREQ  = 25_000_000;
    localparam int BAUD_RATE = 115_200;
endpackage

// In any module:
import uart_pkg::*;
uart_state_t state;            // type from package
Single source of truth. Change the baud rate in one file, all modules update.
Packages provide a single source of truth for shared types and constants. Change once, used everywhere. Eliminates duplicate localparam definitions.

Pre-Class Self-Check

Q1: What does logic replace? What’s its one restriction?

logic replaces both wire and reg. Restriction: single driver only. This catches accidental multi-driver bugs.

Q2: Missing default in always_comb — what happens?

Compile error — latch would be inferred, and always_comb forbids latches. Verilog’s always @(*) silently infers the latch.

Pre-Class Self-Check (cont.)

Q3: Rewrite localparam S_IDLE=2’b00, S_RUN=2’b01, S_DONE=2’b10; reg [1:0] state; as SV enum.

typedef enum logic [1:0] { S_IDLE, S_RUN, S_DONE } my_state_t; my_state_t state;

Q4: What is a packed struct? Why is it synthesizable?

A packed struct stores fields as a contiguous bit vector. An 8-bit data + valid + busy struct = 10-bit vector. Synthesis tools map it to wires/regs identically.
